Drop your logo file here as:

  assets/logo.png

This PNG is used in:
  • Navbar on all 3 pages  (36px wide)
  • Left panel on spin.html (110px wide)
  • Footer on index.html    (20px wide)

The file scales automatically via height:auto in CSS.
For retina displays, use a 2x resolution PNG (e.g. 220px wide).

You can also add:
  assets/logo@2x.png  → for retina srcset (optional)
  assets/favicon.ico  → browser tab icon (optional)
